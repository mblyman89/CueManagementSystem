# COMPATIBILITY NOTES:

# - RPi.GPIO only works on Raspberry Pi hardware

# - Some audio backends may require system-level dependencies (ffmpeg, libsndfile)



# SYSTEM DEPENDENCIES (install via system package manager):

# - ffmpeg (for audio format support)

# - libsndfile (for soundfile library)

# - portaudio (for pyaudio on Windows/Linux)

# - On macOS: Install via Homebrew: brew install ffmpeg libsndfile portaudio

# - On Ubuntu/Debian: sudo apt-get install ffmpeg libsndfile1 portaudio19-dev

# - On Raspberry Pi: sudo apt-get install ffmpeg libsndfile1 portaudio19-dev python3-rpi.gpio
